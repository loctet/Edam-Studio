EDAM Studio — Documentation

Iterative, navigable documentation for all files, functions, and scripts.

Overview

EDAM Studio transforms EDAM (Extended Data-Aware Machines) specifications into Solidity smart contracts with automated test generation. This documentation provides:

  • Python API — Routes, request handlers, parameters
  • Code Generation — Process flow, OCaml generator, Solidity generator, base classes
  • Frontend — React components, props, state, functions
  • OCaml Engine — All .ml files, types, functions with parameters

Flow: EDAM → Code → Test

EDAM (GUI/CLI/API) → payload_generator.js → output.json
     → process_models() → OCamlCodeGenerator.generate_code()
     → ocaml <file>.ml → Python EDAM string
     → ContractCodeGenerator → SolidityGenerator
     → OCamlCodeGenerator.generate_test_code() → cmd_run.sh
     → ./generate_test → Hardhat test JS
     → create_zip_file() → ZIP download
